# litools

litools 是一个基于 Rust / Tauri / SolidJS 的桌面效率工具平台，目标是做成 uTools 风格的本地启动器，并逐步扩展为插件驱动的本地工具生态。


## Roadmap

### Phase 1：本地启动器 MVP

- [x] 面板交互：全键盘操作、搜索、选择、执行
- [x] 原生窗口：无边框、透明背景、失焦隐藏、快捷键唤起
- [x] 系统应用搜索：索引 /Applications、解析 .app、获取图标
- [x] 内置命令：设置、诊断、主题切换、重建索引等
- [x] 基础体验：真实图标、关键词高亮、模糊匹配、最近使用优先

### Phase 2：体验优化 & 插件架构完善（当前）

- [ ] 优化现有功能体验，修复边缘场景和交互细节
- [ ] 完善插件架构设计：统一插件加载、生命周期、权限模型
- [ ] 实现插件开发 SDK（TypeScript），支持本地开发与调试
- [ ] 优化启动器搜索与排序体验

### Phase 3：内置插件 & 系统能力拓展

- [ ] 用户设置插件：配置项管理、快捷键自定义、主题/外观
- [ ] 插件管理插件：已安装列表、启用/禁用、权限查看、安装/卸载
- [ ] 系统能力拓展：剪贴板读写、文件索引增强
- [ ] 启动器能力增强：拼音匹配、更多搜索 Provider

### Phase 4：首个外部插件

- [ ] 剪贴板历史插件：监听、存储、搜索、复用剪贴板内容
- [ ] 验证插件 SDK、权限模型、分发流程的完整链路

### 后续方向

- [ ] 更多系统效率插件（计算器、颜色取色、快捷片段等）
- [ ] 插件 Marketplace 与分发体系
- [ ] 测试覆盖与 CI
- [ ] macOS 签名与分发
